Note

A delimiter is required if either the button filter or the item have more than one value being compared.

Example

data-tags contains "5" works without a delimiter because:

filter.by.attributeValues( this, 'data-tags', '5' ) "5" is the only value being compared.

<article data-tags="5"> "5" is the only value being compared.

If you change either value to "5, 1" it will not work without defining the delimiter, which in this case is ", " (comma + space).
